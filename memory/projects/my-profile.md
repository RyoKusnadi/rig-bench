# my-profile

Next.js 15 + TypeScript + Tailwind personal portfolio website.

## Stack
- Next.js 15, App Router, TypeScript, Tailwind CSS
- Path alias: `@/*` → `src/*`
- GitHub API and Medium RSS fetched at runtime (not build time) — components are client components
- Three.js for 3D visuals in the Hero section

## Key directories
- `src/app/` — all pages (App Router)
- `src/components/` — shared components

## Dev commands
```bash
npm run dev      # dev server, port 3000
npm run build    # production build
npm run lint     # ESLint
```
